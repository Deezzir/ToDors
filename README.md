# Todo

A terminal based Todo list manager written in Rust. My first Rust project.

## Quick Start

```bash
cargo run
```

## Controls

| Key                                                 | Descritption       |
|-----------------------------------------------------|--------------------|
| <kbd>w/↑</kbd>,<kbd>s/↓</kbd>                       | Move UP/DOWN       |
| <kbd>ENTER</kbd>                                    | Action             |
| <kbd>Tab</kbd>                                      | Switch between Todos and Dones |
| <kbd>q/ESC</kbd>                                    | Quit               |
